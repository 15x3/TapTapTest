## 🎯 使用方法

### 方法1：直接替换（推荐）

```bash
# 1. 备份原文件（已完成）
# chat_scenarios_backup.csv
# chats_backup.csv（如果有）

# 2. 替换文件
# 将 chat_scenarios_game.csv 重命名为 chat_scenarios.csv
# 将 chats_game.csv 重命名为 chats.csv

# 3. 启动游戏
# 运行主程序，点击【新手教程】开始
```

### 方法2：修改代码加载

在 `DingtalkData.lua` 中：

```lua
-- 原始代码
local _, rows = loadCSV("data/chat_scenarios.csv")

-- 修改为
local _, rows = loadCSV("data/chat_scenarios_game.csv")
```

同样修改 `chats.csv` 的加载路径。

## 🔧 技术实现

### CSV数据结构

```csv
id,chat_match,delay,type,sender,text,next,options,timeout,default_next,tag,thresholds,require_tag
tut_1,新手教程,0,message,系统,欢迎来到... ,tut_2,,,,,新手
tut_37,新手教程,0,choice,系统,请选择：,start_game,继续练习>tut_retry;开始游戏>start_game,10,,新手
start_game,新手教程,0,set_time,08:00,game_start_2,,,,,新手
game_start_2,新手教程,0,freeze_time,off,game_end,,,,,新手
```

### 标签系统

```lua
-- 玩家选择"立即转发"
pn_forward_done2,*,1.5,message,系统,✓ 已成功转发！,pn_check_others,,,信息筛选

-- 玩家选择"稍后处理"
pn_ignore1,*,0,message,系统,你选择暂时忽略... ,pn_check_others,,,时间压力
```

### 时间控制

```lua
-- 新手教程：时间冻结
game_start_2,新手教程,0,freeze_time,on,tut_37,,,,,

-- 开始游戏：设置时间并解冻
start_game,新手教程,0,set_time,08:00,game_start_2,,,,
```

## 🐛 已知问题

### 1. 时间流速
当前设置为 1:1（10分钟真实时间 = 10分钟游戏时间）。如果觉得太慢/太快，可以：

```lua
-- 在 main.lua 中修改 GameTime.Init()
-- 或添加时间倍率设置
GameTime.SetTimeScale(2.0)  -- 2倍速
```

### 2. 标签显示
游戏结束后，标签只在控制台显示。可以添加UI界面展示：

```lua
-- 在游戏结束时创建UI面板
function ShowTags()
    local tags = TagManager.GetAll()
    -- 创建显示面板
end
```

### 3. 消息同步
不同群的消息通过 `chat_match = "*"` 触发，可能在任意群显示。建议：

```lua
-- 在 DingtalkData.lua 中修改
-- 只在特定群显示跨群消息
```
